# Приложение для древовидного меню.

### Реализована на Python + Django

## Инструкция по установке и запуску:

```
0. Создайте виртуальное окружение: python -m venv venv
1. git clone https://github.com/ThreeToads/TreeMenu
2. pip install django
3. python manage.py migrate 
4. python manage.py createsuperuser
5. python manage.py runserver
```

### Проверка работоспособности:

```
По маршрутам:
http://127.0.0.1:8000/ # Добавлено несколько тестовых страниц для проверки работоспособности
http://127.0.0.1:8000/admin/
```
